# Docs

Lägg djup-docs per ämne här. Exempel:

- `docs/INFRA.md` — tech stack, deployment, databas
- `docs/API.md` — endpoints, requests, responses
- `docs/ARCHITECTURE.md` — högnivå-struktur

Dela inte upp för tidigt. Vänta tills CLAUDE.md börjar kännas för stor — då bryt ut djupare innehåll till en docs/-fil. CLAUDE.md ska peka på vilken docs-fil som är auktoritativ för vilket område.

Ta bort denna README och skapa egna när du är redo.
